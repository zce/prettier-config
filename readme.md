# @zce/prettier-config [![Build Status][actions-img]][actions-url]

> My personal [Prettier](https://prettier.io/) config.

## Install

```bash
npm i @zce/prettier-config -D
```

## Usage

In `package.json`:

```json
{
  "prettier": "@zce/prettier-config"
}
```

Or `.prettierrc`:

```json
"@zce/prettier-config"
```
